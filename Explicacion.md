Este proyecto es una API REST desarrollada con Node.js y Express para gestionar opiniones como por ejemplo comentario en publicaciones.
El sistema permite manejar operaciones típicas como autenticación, validación, control de roles y almacenamiento de datos en MongoDB.
El funcionamiento del API empieza cargando las variables de entorno desde el archivo .env, luego tenemos que importar dependencias, se realiza la conexión con la base de datos en MongoDB, se inicia el servidor en el puerto que definimos.
La estructura de la aplicación es de forma modular, separando configuraciones, middlewares, rutas y controladores. Las rutas reciben las solicitudes del cliente y las dirigen hacia el controlador que corresponden, en donde se ejecuta la lógica del negocio. Los modelos definen la estructura de los datos que se almacenan en la base de datos.
El proyecto demuestra la implementación de un backend moderno orientado a servicios, utilizando principios de separación de responsabilidades y configuración por entorno. Está preparado para escalar, integrarse con otras aplicaciones y operar en entornos de desarrollo o producción. En conjunto, el sistema gestor de opiniones no solo permite administrar información mediante APIs, sino que también integra prácticas profesionales de seguridad, estructura y manejo de errores que lo hacen adecuado para aplicaciones reales.
